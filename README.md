ProjectOps Dashboard is a project management case study application built with React and TypeScript that simulates how teams track and communicate delivery for a website redesign and client portal launch. 

The application is structured around real-world PM workflows, including project overview, stakeholder summary, task tracking with filtering, milestone timeline management, risk register tracking, and executive-style weekly status reporting. 

This project was designed to demonstrate how project status, scope, risks, and stakeholder communication can be organized into a clear, usable interface that supports delivery visibility and decision-making.
