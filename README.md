ProjectOps Dashboard is a project management case study application built with React and TypeScript to simulate end-to-end delivery tracking for a website redesign and client portal launch. The app includes project overview, stakeholder summary, dashboard reporting, task tracking with filters, milestone timeline management, risk register tracking, and executive-style weekly status reporting. It was designed to reflect practical PM workflows, improve delivery visibility, and demonstrate cross-functional coordination, risk mitigation, and stakeholder communication.
